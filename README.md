Steam - онлайн-платформа для распространения компьютерных игр и программного обеспечения. 

Бизнес-цель: оптимизация маркетинговой стратегии и позиционирования игр на платформе Steam для максимизации уровня вовлеченности пользователей и монетизации платформы



Итоговый результат будет содержать систему рекомендаций для пользователе. основанный на анализе, сделанным нами ранее.

Информация об собранных данных: у нас есть несколько датасетов

1. Общие данные по играм для 2000-2025 годов
2. Дополнительные данные для 23-25 годов
3. Финансовые показатели со Steam
